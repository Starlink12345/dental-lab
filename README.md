# Dental Lab — лендинг "Звёздный техник"

Короткое описание
Небольшой одностраничный сайт (HTML + CSS) для зубного техника. Демонстрация услуг лаборатории, контактов и быстрых кнопок связи (звонок, WhatsApp/Telegram).

Онлайн-демо
- https://starlink12345.github.io/dental-lab/

Структура репозитория
- index.html — главная страница (HTML)
- style.css — стили (CSS)
- main-foto.jpg — пример изображения для Hero (опционально)

Как запустить локально
1. Открой папку с проектом.
2. Двойной клик по файлу `index.html` — страница откроется в браузере.
(Если хочешь — можно поднять локальный сервер, но для простого лендинга это не требуется.)

Как править
- Редактируй `index.html` и `style.css` в репозитории, затем делай Commit.
- Для добавления фото — загружай файл в корень репо и обновляй путь в `index.html`.

Развертывание (GitHub Pages)
- Сайт уже настроен через GitHub Pages. Если нужно пересобрать, зайди: Settings → Pages → убедись, что выбрана ветка `main` и корень (`/root`), затем сохрани.
